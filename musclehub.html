We'd like to download a giant DataFrame containing all of this data. You'll need to write a query that does the following things:

  1. Not all visits in visits occurred during the A/B test. You'll only want to pull data where visit_date is on or after 7-1-17.

  2. You'll want to perform a series of LEFT JOIN commands to combine the four tables that we care about. You'll need to perform the joins on first_name, last_name, and email. Pull the following columns:

  • visits.first_name
  • visits.last_name
  • visits.gender
  • visits.email
  • visits.visit_date
  • fitness_tests.fitness_test_date
  • applications.application_date
  • purchases.purchase_date

Save the result of this query to a variable called df.

Hint: your result should have 5004 rows. Does it?

In [44]:
df = sql_query('''
SELECT visits.first_name, 
  visits.last_name,  
  visits.visit_date, 
  visits.gender, 
  visits.email,
  fitness_tests.fitness_test_date, 
  applications.application_date,
  purchases.purchase_date
FROM visits
LEFT JOIN fitness_tests
  ON fitness_tests.first_name = visits.first_name 
  AND fitness_tests.last_name = visits.last_name
  AND fitness_tests.email = visits.email
LEFT JOIN applications
  ON  applications.first_name = visits.first_name
  AND applications.last_name = visits.last_name
  AND applications.email = visits.email
LEFT JOIN purchases
  ON purchases.first_name = visits.first_name
  AND purchases.last_name = visits.last_name
  AND purchases.email = visits.email
WHERE visits.visit_date >= '7-1-17'
''')

print(len(df))

It looks like more people from Group B turned in an application. Why might that be?

We need to know if this difference is statistically significant.

Choose a hypothesis tests, import it from scipy and perform it. Be sure to note the p-value. Is this result significant?

In [54]:
from scipy.stats import chi2_contingency

# Contingency table
#         Application |  No Application
# ----+------------------+------------
# Group A | 250       |  2254
# Group B | 325       |  2175

#The null hypothesis here is that 
#there's no significant difference 
#between the datasets. 

X = [[250, 2254],
     [325, 2175]]
chi2, pval, dof, expected = chi2_contingency(X)
print(pval)

0.0009647827600722304

We have to use the Chi Square Test for testing our null hypothesis above: there's significant difference between two datasets if we get a p-value less than 0.05.

The p-value is == 0.0009647827600722304. That is, we can say that the results of the A/B test ARE significantly different and should be taken into consideration.

This means, that potential customers of the MuscleHub are more likely to complete the application process if they don't have to go through the initial intro training session.

from scipy.stats import chi2_contingency

# Contingency table
#         Member |  No Member
# ----+------------------+------------

# Group B | 250       |  75

#The null hypothesis here is that 
#there's no significant difference 
#between the datasets. 

X = [[200, 50],
     [250, 75]]
